# MLB Pitching Performance Analysis

## Overview

This project analyzes Major League Baseball (MLB) pitching performance using both traditional and advanced statistics. The goal is to identify which metrics are most strongly associated with ERA and overall pitcher effectiveness.

## Research Question

Which pitching metrics best explain variation in ERA, and how useful are advanced statistics compared to traditional measures?

## Data

The dataset includes MLB pitcher performance statistics such as:

* ERA (Earned Run Average)
* Strikeout rate (K%)
* Walk rate (BB%)
* Whiff rate
* Hard-hit rate
* Barrel rate
* wOBA (Weighted On-Base Average)
* xwOBA (Expected wOBA)

## Methodology

The analysis follows a structured data analytics workflow:

* Data cleaning and preprocessing
* Exploratory data analysis (EDA)
* Correlation analysis between ERA and pitching metrics
* Data visualization (scatterplots, trends)
* Regression modeling to evaluate predictive relationships

## Tools & Technologies

* Python
* pandas
* numpy
* matplotlib
* seaborn
* statsmodels
* Jupyter Notebook

## Key Insights

* Strikeout rate (K%) shows a strong negative relationship with ERA
* Hard-hit rate and xwOBA are strong indicators of pitching performance
* Advanced metrics provide more insight than traditional stats alone
* Multiple factors contribute to ERA, making it imperfect as a standalone measure

## Example Outputs

The project includes:

* Visualizations of ERA vs key pitching metrics
* Correlation analysis across variables
* Regression results interpreting the impact of each statistic
